# My Personal Portfolio

This is my personal portfolio app built using NEXT.js. The app showcases my skills and experience as a developer. The app uses various dependencies such as Framer Motion, TailwindCSS, and others.

## Installation

### To get started with the app, follow these steps:

1. Clone this repository to your local machine.
2. Install dependencies using npm install or yarn install.
3. Start the development server using npm run dev or yarn dev.
4. Open your browser and go to http://localhost:3000.

## Features

### My Personal Portfolio provides the following features:

- Home section showcasing my profile and summary.
- Skillset section showcasing my skillset.
- Projects section showcasing my personal and professional projects.
- Certificates section showcasing my certificates.
- Contact section allowing visitors to send me a message.

## Dependencies
